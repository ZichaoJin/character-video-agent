# 一二 角色参考图

请将一二（IP 角色）的**官方或指定画风**参考图放在本目录下。

**多角度（推荐）**：可放四张图，便于生成时更一致  
- **front.png**（或 .jpg）正面  
- **oblique.png** 斜侧面  
- **side.png** 侧面  
- **back.png** 背面  

**单张**：若只放一张，命名为 **best.png**（或 best.jpg）即可。

以上图片会由 GPT-4V 生成「角色+画风」描述，用于 DALL·E 每镜生成；ROICtrl/Mix-of-Show 也会使用本目录。请使用一二 IP 的立绘或设定图，不要用真人照片。
